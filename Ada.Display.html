    cycle_spec() View Source 
    cycle_spec() :: {:cycle, [{Ada.Display.Driver.buffer(), duration_ms()}]}

    Represents a specification for a cyclic screen, which loops over different buffers, each one with a specific duration.

    For example:

    [
  {Ada.UI.Helpers.chars_to_matrix('A   '), 200},
  {Ada.UI.Helpers.chars_to_matrix(' A  '), 200},
  {Ada.UI.Helpers.chars_to_matrix('  A '), 200},
  {Ada.UI.Helpers.chars_to_matrix('   A'), 200}
]
