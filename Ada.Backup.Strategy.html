    ada v0.1.0 Ada.Backup.Strategy behaviour View Source

    The Ada.Backup.Strategy behaviour defines a module capable of backing up and restoring files from a specific provider.

    Link to this section Summary

    Types

    contents()

    The contents of the backup file

    name()

    The name to use when saving the file

    The path where a backup is stored

    Callbacks

    configured?()

    This callback should check if the module is properly setup, e.g. if any access token is present.

    download_file(path)

    Download a file at a given path, returning its contents.

    list_files()

    Returns a list of paths where backups are stored. Each one of these paths should be compatible with the download_file/1 callback.

    upload_file(name, contents)

    Uploads a file with the specified contents under the given name, returning its path.

    configured?() View Source 
    configured?() :: boolean()

    This callback should check if the module is properly setup, e.g. if any access token is present.

    The function will be invoked at application boot. If it returns false, backups will be disabled.
