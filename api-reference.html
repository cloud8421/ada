    ada v0.1.0 API Reference

    Modules

    Ada.Backup.Strategy

    The Ada.Backup.Strategy behaviour defines a module capable of backing up and restoring files from a specific provider.

    Ada.Backup.Uploader

    Controls the upload of the database file to the configured backup location.

    Ada.CLI.FishCompletion

    Provides autocomplete settings for the Fish shell.

    Ada.CLI.Helpers

    Provides helper functions to ease the composition of CLI tasks.

    Ada.CLI.Markup

    This module defines semantic helpers that can be used to format CLI-based reports.

    Ada.CRUD

    This module collects generalised ways to manage database entries.

    Ada.Display

    Supports the following contents

    Ada.Display.Driver

    This behaviour represents a generic display driver, which needs to be implemented when supporting a new LED display or similar.

    Ada.Email

    Represents an email that can be sent via different adapters.

    Ada.Email.Adapter

    An email adapter takes a Ada.Email.t/0 and sends it, reporting the result.

    Ada.Email.Adapter.Sendgrid

    Implements the Ada.Email.Adapter behaviour on top of the Sendgrid API (documentation available at https://sendgrid.com/docs/API_Reference/api_v3.html).

    Ada.Email.Quickchart

    Minimal DSL and functions to create image charts via https://quickchart.io.

    Ada.HTTP.Client

    Simple http client based on httpc.

    Ada.Preferences

    Control system wide preferences.

    Ada.PubSub

    Provides a single-node pub-sub infrastructure on top of Registry.

    Ada.Schema.Frequency

    The module expresses the idea of something that can be repeated at regular intervals.

    Ada.Schema.Location

    Represents a location (e.g. home or office).

    Ada.Schema.Preference

    A preference is a pair of key, value settings which affect the behaviour of the entire application.

    Ada.Schema.ScheduledTask

    Represents a boilerplate for the recurring execution of a workflow.

    Ada.Schema.User

    Represents a system user, identified by a numeric ID.

    Ada.Workflow

    The Ada.Workflow module specifies a behaviour which needs to be implemented by all workflows.

    Logger.Backends.Telegraf

    Telegraf compatible, syslog formatted Logger backend.
